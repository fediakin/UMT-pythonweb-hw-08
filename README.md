# Contacts REST API 📇

REST API для зберігання та управління контактами. Проєкт розроблено як виконання домашнього завдання (Тема 8) з використанням сучасного стеку веб-розробки на Python.

## 🚀 Стек технологій

- **Фреймворк:** FastAPI
- **База даних:** PostgreSQL
- **ORM:** SQLAlchemy 2.0 (використання сучасного синтаксису `Mapped` та `select()`)
- **Валідація:** Pydantic V2
- **Сервер:** Uvicorn

## 📂 Структура проєкту

Проєкт побудовано за модульною архітектурою для зручного масштабування:

- `app/core/` — налаштування підключення до бази даних (`database.py`).
- `app/models/` — SQLAlchemy моделі бази даних (`contact.py`).
- `app/schemas/` — Pydantic схеми для валідації вхідних та вихідних даних.
- `app/routers/` — логіка маршрутизаторів (CRUD-операції та пошук).
- `app/main.py` — точка входу додатку.

## ✨ Основні функції (CRUD+)

1. **Створення контакту:** з перевіркою на унікальність email та валідацією довжини полів і дати народження (дата не може бути в майбутньому).
2. **Отримання списку:** з підтримкою фільтрації (Query-параметри) за `first_name`, `last_name` та `email` (часткове співпадіння).
3. **Отримання одного контакту:** за унікальним ідентифікатором (ID).
4. **Оновлення:** з перевіркою конфліктів email при зміні.
5. **Видалення:** за ID.
6. **Дні народження:** Спеціальний ендпоінт `/contacts/birthdays` для отримання списку контактів, у яких день народження припадає на найближчі 7 днів (з урахуванням високосних років).
